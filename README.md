## Quick Start

- Install [YC cli](https://cloud.yandex.com/docs/cli/quickstart)
- Run commands:

```
yc init
```

```
yc config list
```

```
cp ./terraform/terraform.tfvars_ ./terraform/terraform.tfvars
```

- Provide `token`, `cloud_id`, `folder_id` into the file `./terraform/terraform.tfvars`

```
npm run init
```

```
npm run deploy
```
